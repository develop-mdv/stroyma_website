# STROYMA - Интернет-магазин строительных материалов

## О проекте

STROYMA - это веб-сайт для магазина строительных материалов, разработанный на Django. Система позволяет управлять каталогом товаров, обработкой заказов, услугами и взаимодействием с клиентами.

## Технологии

- Python 3.x
- Django 5.2
- PostgreSQL
- Bootstrap (для фронтенда)
- Jazzmin (для улучшенного интерфейса админ-панели)

## Установка и настройка

### Требования

- Python 3.8 или выше
- PostgreSQL
- Git (для клонирования репозитория)

### Шаги по установке

1. **Клонирование репозитория**
```bash
git clone <URL_репозитория>
cd stroyma_website
```

2. **Создание виртуального окружения и установка зависимостей**
```bash
python -m venv .venv
# Для Windows:
.venv\Scripts\activate
# Для Linux/Mac:
source .venv/bin/activate
pip install -r requirements.txt
```

3. **Настройка переменных окружения**
Создайте файл `.env` в корневой директории проекта со следующим содержимым:
```
SECRET_KEY=ваш_секретный_ключ
DB_NAME=имя_базы_данных
DB_USER=пользователь_базы_данных
DB_PASSWORD=пароль_базы_данных
DB_HOST=хост_базы_данных
DB_PORT=порт_базы_данных
EMAIL_HOST_USER=ваш_email
EMAIL_HOST_PASSWORD=пароль_от_email
```

4. **Создание базы данных**
```bash
# Подключитесь к PostgreSQL
psql -U postgres
# Создайте базу данных
CREATE DATABASE имя_базы_данных;
# Выйдите из PostgreSQL
\q
```

5. **Применение миграций и создание суперпользователя**
```bash
python manage.py migrate
python manage.py createsuperuser
```

6. **Запуск сервера разработки**
```bash
python manage.py runserver
```

Сайт будет доступен по адресу http://127.0.0.1:8000, а админ-панель по адресу http://127.0.0.1:8000/admin/

## Структура проекта

- **accounts**: Управление пользователями и профилями
- **products**: Управление товарами, категориями, заказами и корзиной
- **services**: Управление услугами и фотографиями/видео услуг
- **templates**: Шаблоны сайта
- **static**: Статические файлы (CSS, JS, изображения)
- **media**: Загруженные пользователями файлы

## Функционал сайта

### Каталог товаров

- Иерархические категории товаров (с использованием MPTT)
- Детальная страница товара с фотографиями, описанием и ценой
- Фильтрация товаров по категориям, цене и другим параметрам
- Поиск товаров

### Корзина и заказы

- Добавление товаров в корзину с указанием количества
- Просмотр и редактирование корзины
- Оформление заказа с указанием данных доставки
- Отслеживание статуса заказа
- История заказов в личном кабинете

### Услуги

- Каталог услуг с подробным описанием
- Фотографии и видео примеров работ
- Возможность заявки на услугу

### Пользователи и личный кабинет

- Регистрация и авторизация пользователей
- Личный профиль с возможностью редактирования
- Подтверждение email
- История заказов
- Сохранение адресов доставки

## Инструкция по работе с админ-панелью

### Вход в админ-панель

1. Перейдите по адресу http://ваш_домен/admin/
2. Введите логин и пароль администратора

### Управление товарами

#### Добавление нового товара:

1. В меню выберите "Товары" > "Добавить"
2. Заполните следующие поля:
   - Название товара
   - URL-имя (генерируется автоматически, но можно изменить)
   - Описание
   - Цена
   - Количество на складе
   - Рейтинг (от 1 до 5)
   - Изображение
   - Категории (можно выбрать несколько)
   - SEO информация (заголовок, описание, ключевые слова)
3. Нажмите "Сохранить"

#### Добавление дополнительных изображений товара:

1. На странице редактирования товара прокрутите вниз до раздела "Изображения товара"
2. Нажмите "Добавить еще одно Изображение для..."
3. Загрузите изображение, при желании добавьте название и порядок отображения
4. Нажмите "Сохранить"

#### Управление категориями товаров:

1. В админ-панели выберите "Категории"
2. Для добавления новой категории нажмите "Добавить категорию"
3. Заполните название, URL-имя и выберите родительскую категорию (если это подкатегория)
4. Для перемещения категорий используйте drag-and-drop интерфейс на странице списка категорий

### Управление заказами

#### Просмотр и редактирование заказов:

1. В меню выберите "Заказы"
2. Для просмотра деталей заказа кликните на номер заказа
3. На странице заказа вы увидите:
   - Информацию о пользователе
   - Список товаров с количеством и суммой
   - Общую стоимость заказа
   - Историю статусов заказа

#### Изменение статуса заказа:

1. Откройте заказ для редактирования
2. В поле "Статус" выберите один из вариантов:
   - В ожидании (pending)
   - Обработка (processing)
   - Доставляется (shipped)
   - Завершен (completed)
   - Отменен (canceled)
3. Нажмите "Сохранить"

При изменении статуса заказа автоматически создается запись в истории статусов.

#### Экспорт заказов:

На странице списка заказов вы можете:
1. Выбрать заказы, установив флажки
2. В выпадающем меню "Действие" выбрать:
   - "Экспорт в CSV" для получения табличного формата
   - "Экспорт в PDF" для более форматированного отчета

### Аналитика и отчеты

1. В меню выберите "Аналитика" > "Отчет по продажам"
2. На странице отчета вы увидите:
   - График продаж за выбранный период (день, неделя, месяц)
   - Статистику по количеству заказов и общей сумме
   - Популярные товары
   - Статистику по статусам заказов

Используйте фильтры для изменения периода и получения более детальной информации.

### Управление услугами

#### Добавление услуги:

1. В меню выберите "Услуги" > "Добавить"
2. Заполните информацию:
   - Название услуги
   - Краткое и подробное описание
   - Загрузите изображение/иконку
   - Укажите преимущества (по одному на строку)
   - Заполните SEO информацию
3. Нажмите "Сохранить"

#### Добавление фотографий и видео к услуге:

1. На странице редактирования услуги прокрутите до раздела фотографий или видео
2. Нажмите "Добавить еще одно Фото услуги" или "Добавить еще одно Видео услуги"
3. Загрузите файл, укажите название, описание и порядок отображения
4. Нажмите "Сохранить"

### Работа с клиентами

#### Просмотр информации о клиентах:

1. В меню выберите "Профили пользователей"
2. В списке показана основная информация о пользователях
3. Кликните на профиль для просмотра подробной информации

#### Поиск клиента:

1. В верхней части админ-панели есть поле поиска
2. Введите имя пользователя, email или телефон
3. В результатах поиска выберите нужного клиента

#### Просмотр заказов клиента:

1. Откройте профиль клиента
2. В разделе "История заказов" будут показаны все заказы пользователя
3. Кликните на номер заказа для перехода к подробной информации

## Дополнительная информация

### Рекомендации по SEO-оптимизации

- Всегда заполняйте поля "SEO заголовок" и "SEO описание" для категорий, товаров и услуг
- Используйте ключевые слова в названиях и описаниях
- Добавляйте альтернативный текст к изображениям

### Управление цветами и текстурами

Система поддерживает управление цветами и текстурами для фасадов:
1. В меню выберите "Цвета фасадов" или "Базовые текстуры"
2. Добавьте новые цвета с кодом HEX или загрузите текстуры
3. Эти параметры можно будет использовать при создании товаров

### Оптимизация производительности

- Система использует кеширование для популярных товаров и аналитики
- Для больших каталогов рекомендуется настроить дополнительный кеш (Redis/Memcached)
- Регулярно проверяйте наличие индексов в базе данных для часто используемых запросов

## Устранение неполадок

### Проблемы с загрузкой изображений

1. Проверьте права доступа к директории media/
2. Убедитесь, что в настройках правильно указаны MEDIA_URL и MEDIA_ROOT

### Ошибки в работе с базой данных

1. Проверьте подключение к базе данных
2. Убедитесь, что все миграции применены
3. Выполните команду `python manage.py check` для проверки ошибок в моделях

### Проблемы с отправкой email

1. Проверьте настройки SMTP в файле settings.py
2. Убедитесь, что указаны правильные учетные данные для почтового сервера
